# ELIZA 3D

## Overview
ELIZA is an early natural language processing program that simulates conversation by using pattern matching and substitution methodology. It was created in the 1960s and has since inspired various implementations and adaptations beyond its initial scope. ELIZA 3D is a modern interpretation that adds a three-dimensional and interactive layer to the experience.

## How ELIZA Works
The core of ELIZA's functionality lies in its ability to recognize certain keywords within the input from users. It utilizes a scripting system where specific phrases and words trigger predefined responses. The program analyzes user input, looks for key terms or phrases, and then constructs responses that often resemble a real conversation, making it seem as if the program understands the context.

### The Process:
1. **Input Analysis:** ELIZA analyzes the text entered by the user.
2. **Response Generation:** Using predefined rules, ELIZA generates an output based on the analyzed input.
3. **User Interaction:** The user receives the response and can continue the dialogue, creating an ongoing conversation.

## How to Use ELIZA 3D
1. **Start the Application:** Launch the ELIZA 3D application.
2. **Enter Your Text:** Type your message in the input box provided.
3. **Receive Responses:** ELIZA will process your message and generate a response within the interactive 3D environment.
4. **Continue the Conversation:** You can respond back and forth to explore various dialogues and scenarios.

## Conclusion
ELIZA 3D not only retains the classic charm of the original ELIZA program but also enhances the conversational experience through immersive 3D graphics. It’s a fascinating exploration of AI and human interaction, perfect for anyone interested in the field of artificial intelligence.
